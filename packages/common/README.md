# @typescript-auth/common ☼
The main propose of this package, is to provide general interfaces, types for authorization & authentication.
It also contains core functions, for few other modules of the `@typescript-auth` namespace.

**Table of Contents**

- [Installation](#installation)
- [Usage](#usage)

## Installation

```bash
npm install @typescript-auth/common --save
```
